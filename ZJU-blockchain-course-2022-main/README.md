⬆ 可以️修改成你自己的项目名。



## 如何运行

1. 在本地启动ganache应用。

2. 使用Rimx部署代码到本地
   复制项目MyERC20.sol,StudentSocietyDAO.sol，编译、运行代码

3. 修改合约地址、本地节点:复制合约地址与abis到abis./contract-addresses.json

4. 在 `./front` 中启动前端程序，运行如下的命令：
    npm run start

## 功能实现分析
实现了智能合约的撰写
实现了代码在区块链上的部署
实现了用户界面登陆与本地网的链接与基本功能
